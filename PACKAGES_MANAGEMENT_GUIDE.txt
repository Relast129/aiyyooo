================================================================================
                            ADD NEW PACKAGE
================================================================================

1. Fill in the form:
   
   **Required Fields:**
   - Package Name: e.g., "Summer Beach Paradise"
   - Duration: e.g., "5 Days / 4 Nights"
   - Current Price: e.g., "$699"
   - Description: Brief overview
   - Highlights: One per line (what makes it special)
   - What's Included: One per line (what's in the package)
   
   **Optional Fields:**
   - Original Price: e.g., "$899" (shows discount)
   - Discount Badge: e.g., "30% OFF"
   - Badge: e.g., "Best Seller", "New", "Limited Time"
   - Season: e.g., "Summer Only", "Winter Special", "All Year"
   - Status: Active (visible) or Inactive (hidden)

2. Click "Add Package"
3. Package appears on packages page immediately!

================================================================================
                            ACTIVATE / DEACTIVATE
================================================================================

**Deactivate (Hide):**
- Package stays in database
- Not visible on website
- Can reactivate anytime
- Perfect for seasonal packages

**Activate:**
- Makes hidden package visible again
- All data preserved
- Appears on website immediately

**When to use:**
- Off-season packages
- Testing new packages
- Temporarily sold-out packages
- Packages under revision

================================================================================
                            TROUBLESHOOTING
================================================================================

**Problem:** Changes not appearing on packages page
**Solution:** 
1. Check package is "Active"
2. Refresh packages page (Ctrl+F5)
3. Check browser console for errors

**Problem:** Can't edit package
**Solution:**
1. Make sure you're logged in as admin
2. Refresh admin dashboard
3. Try again

**Problem:** Package deleted by mistake
**Solution:**
Unfortunately, deletions are permanent. Always use "Deactivate" instead of "Delete" unless you're absolutely sure.

**Problem:** Form not submitting
**Solution:**
1. Fill all required fields (marked with *)
2. Check browser console for errors
3. Try refreshing page
