================================================================================
                            DEPLOYMENT INSTRUCTIONS
================================================================================

STEP 1: Install Vercel CLI
---------------------------
Open Command Prompt (Windows) or Terminal (Mac/Linux) and run:

    npm install -g vercel

STEP 2: Deploy to Vercel
-------------------------
Navigate to your project folder:

    cd "c:\Users\ramzy\Downloads\ceyora html"

Login to Vercel (this will open your browser):

    vercel login

Deploy the website:

    vercel

Answer the prompts:
- Set up and deploy? YES
- Which scope? Select your account
- Link to existing project? NO
- Project name? ceyora-holidays
- Directory? ./ (just press Enter)
- Override settings? NO

STEP 3: Create Vercel KV Database
----------------------------------
1. Go to: https://vercel.com/dashboard
2. Click on your "ceyora-holidays" project
3. Click "Storage" tab
4. Click "Create Database"
5. Select "KV" (Key-Value Store)
6. Name it: ceyora-kv
7. Click "Create"

STEP 4: Redeploy (IMPORTANT!)
------------------------------
After creating the database, run:

    vercel --prod

Your website is now LIVE! üéâ

================================================================================
                        HOW TO USE ADMIN DASHBOARD
================================================================================

1. LOGIN
--------
Go to: https://your-domain.vercel.app/admin-login.html
Enter username and password
Click "Login"

2. UPLOAD GALLERY IMAGES
-------------------------
- Click "Gallery Management" tab
- Click "Select Image" button
- Choose image (max 4MB)
- Enter caption/description
- Click "Upload Image"
- Image appears immediately on About page

3. MANAGE REVIEWS
------------------
- Click "Reviews Management" tab

To ADD new review:
- Fill in customer name
- Select country
- Choose rating (1-5 stars)
- Write review text
- Click "Add Review"
- Review appears on Reviews page and Homepage carousel

To APPROVE visitor reviews:
- Scroll to "All Reviews" section
- Find pending reviews
- Click "Approve" button
- Review now shows publicly

To DELETE reviews:
- Find the review
- Click "Delete" button
- Confirm deletion

4. LOGOUT
---------
Click "Logout" button in top-right corner
