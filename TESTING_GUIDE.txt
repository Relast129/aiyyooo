================================================================================
                            COMMON ISSUES & FIXES
================================================================================

Problem: "Missing BLOB_READ_WRITE_TOKEN"
Fix: Create Blob Store in Vercel dashboard and connect to project

Problem: Images upload but don't appear on About page
Fix: 
- Check browser console for errors
- Verify gallery-loader.js is loaded
- Check API response in Network tab

Problem: Reviews don't appear on homepage carousel
Fix:
- Make sure reviews are "approved" status
- Check that reviews-carousel.js is loaded
- Verify call-to-action section exists on homepage

Problem: Data doesn't persist after refresh
Fix:
- Verify Blob Store is connected
- Check Vercel logs for API errors
- Redeploy: vercel --prod

Problem: Upload fails with 500 error
Fix:
- Check Vercel function logs
- Verify Blob Store has available space
- Check image size (keep under 10MB)
