    --primary: #00a896;
    --success: #10b981;

```javascript
const ADMIN_CREDENTIALS = {
    username: 'admin',
    password: 'your-new-password'
};
```

**Add More Stats:**
Edit `js/dashboard.js` in the `updateStats()` function

================================================================================
                            TROUBLESHOOTING
================================================================================

**Problem:** Dashboard not loading
**Solution:**
1. Clear browser cache (Ctrl+Shift+Delete)
2. Check console for errors (F12)
3. Verify you're logged in
4. Try incognito mode

**Problem:** Charts not displaying
**Solution:**
1. Check internet connection (Chart.js CDN)
2. Verify Chart.js is loaded
3. Check browser console
4. Refresh page (Ctrl+F5)

**Problem:** Data not updating
**Solution:**
1. Check API endpoints are working
2. Verify Blob Store connection
3. Check browser console for errors
4. Try re-uploading data

**Problem:** Login redirect not working
**Solution:**
1. Clear session storage
2. Close all browser tabs
3. Try logging in again
4. Check admin-auth.js is loaded
