================================================================================
                            SETUP STEPS (SIMPLE!)
================================================================================

STEP 1: Push Updated Code to GitHub
------------------------------------
git add .
git commit -m "using-blob-store-only"
git push

STEP 2: Create Blob Store in Vercel
------------------------------------
1. Go to: https://vercel.com/dashboard
2. Click your project (aiyyooo)
3. Click "Storage" tab
4. Click "Create Database"
5. Select "Blob" (Blob Store)
6. Name it: ceyora-blob
7. Click "Create"

STEP 3: Connect to Your Project
--------------------------------
1. After creating, click "Connect to Project"
2. Select your project from dropdown
3. Click "Connect"
4. Vercel adds environment variable: BLOB_READ_WRITE_TOKEN

STEP 4: Redeploy
----------------
Vercel will auto-deploy from GitHub, or manually:

vercel --prod

================================================================================
                            TROUBLESHOOTING
================================================================================

Problem: "Missing BLOB_READ_WRITE_TOKEN"
Solution:
1. Create Blob Store in Vercel dashboard
2. Connect it to your project
3. Redeploy: vercel --prod

Problem: Images not uploading
Solution:
1. Check Vercel logs: vercel logs
2. Verify Blob Store is connected
3. Check deployment has latest code

Problem: Reviews not persisting
Solution:
1. Same as above - Blob Store must be connected
2. Check browser console for errors
3. Verify API endpoints are working

Problem: Old data missing after update
Solution:
1. This is normal - Blob Store starts fresh
2. Re-upload images through admin panel
3. Re-add reviews through admin panel
