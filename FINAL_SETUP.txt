================================================================================
                        API ENDPOINTS
================================================================================

/api/gallery
- GET: Fetch all gallery images
- POST: Upload new image (admin only)
- DELETE: Remove image (admin only)

/api/reviews
- GET: Fetch all reviews (or only approved)
- POST: Add new review (admin only)
- PUT: Approve review (admin only)
- DELETE: Remove review (admin only)

/api/packages
- GET: Fetch all packages (or only active)
- POST: Create new package (admin only)
- PUT: Update package (admin only)
- DELETE: Remove package (admin only)
